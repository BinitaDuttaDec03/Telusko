1) Primitive Values:
    a) int - 4 bytes
    b) long - 8 bytes
    c) short - 2 bytes
    d) byte - 1 byte
    e) float - 4 bytes
    f) double - 8 bytes
    g) char - 2 bytes
    e) boolean - true or false (not represented as 1 or 0)

2) To work with a decimal point value, java use double type as default. To explicitly mention that the type is of float, use [float num=1.3f;]
